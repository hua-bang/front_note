### Js文件的引入

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <!-- 外边引入js -->
    <script src="1.js"></script>
    <!-- 文件内部嵌入js -->
    <script>
        console.log("hug")
    </script>
</head>
<body>
    <h2>www</h2>
</body>
</html>
```
