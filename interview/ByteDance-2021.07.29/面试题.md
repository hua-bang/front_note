13. 手写题

    - 爬楼梯问题：爬楼梯时，每一步会有两个选择：爬一个台阶和爬两个台阶，问：楼梯总台阶数为n，则一共有多少种爬法，写一个函数f，使得：总的爬法= f(n)。举例：n=3时，则共有：（1,1,1)、(1,2) 、(2,1)三种爬法，则f(3)=3。

    ```js
    function f(n) {
      let dp = [];
      dp[0] = 1;
      dp[1] = 2;
      for(let i = 2 ; i < n; i++) {
        dp[i] = dp[i-1] + dp[i - 2];
      }
      return dp[n-1];
    }
    ```
